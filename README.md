# `mipt-ded-bardak` -- Игровой движок для многопользовательской 2D онлайн-игры на C++

Данный репозиторий содержит стандарты (в том числе и прилагающиеся к ним заголовочные файлы),
согласованные (или согласуемые) группой на курсе по C/C++ И. Р. Дединского в МФТИ, в 4-м семестре.

Реализации данных стандартов скорее всего можно будет найти у участников на их github профилях.
