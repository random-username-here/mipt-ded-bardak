# `mipt-ded-bardak` -- Игровой движок для многопользовательской 2D онлайн-игры на C++

Данный репозиторий содержит стандарты и общие либы, согласованные (или согласуемые)
группой на курсе по C/C++ И. Р. Дединского в МФТИ, в 4-м семестре.

Реализации данных стандартов скорее всего можно будет найти у участников на их github профилях.

## Обзор имеющегося

 - `binmsg/textmsg` -- Расширяемый стандарт передачи сообщений между клиентом и сервером, был заменён `binmsg`
 - `binmsg/binmsg` -- Стандарт подобный `textmsg`, только бинарный.
 - `modlib/` -- Либа для загрузки .so-шных модулей, по COM модели
 - `modlib-ifc/` -- Заголовочники со стандартными интерфейсами для модулей
 - `proto/` -- Протоколы для общения клиента с сервером с помощью стандарта `binmsg`
